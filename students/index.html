Ученикам

Откройте ссылки по порядку: сначала Code.org, затем материалы курса, установка среды и вход в Informatics.

1. С чего начать

  • Code.org: стартовый раздел

    Откройте раздел и пройдите первые задания перед основным курсом. Кодовое слово для своего имени возьмите у учителя.

2. Материалы курса

  • Модуль 1

    Сначала заведите Gmail. Затем отправьте письмо учителю на ufarobot@gmail.com: напишите имя ученика.

  • M1 Karel: стартовый архив

    Архив для Модуля 1. Скачайте его, распакуйте и откройте всю папку проекта в IntelliJ IDEA.

  • Модуль 2

    Материалы второго модуля на Google Sites. Доступ может открываться после входа в Gmail.

  • Информатикс

    Раздел курса с заданиями на Informatics. Если раньше входили через Gmail, теперь входите по той же почте и паролю, без кнопки Gmail.

  • Модуль 3

    Материалы третьего модуля на Google Sites. Открывайте после первых двух модулей.

3. IntelliJ IDEA

  • IntelliJ IDEA Community Edition: Windows

    Версия для обычных компьютеров на Windows 10/11.

  • IntelliJ IDEA Community Edition: Windows ARM64 / Parallels

    Версия для Windows 11 ARM, например в Parallels на Mac.

  • IntelliJ IDEA Community Edition: Mac Apple Silicon

    Для Mac с процессорами Apple M1, M2, M3, M4 и новее.

  • IntelliJ IDEA Community Edition: Mac Intel

    Для старых Mac с процессором Intel.

4. Informatics

  • Informatics: основной вход

    Если раньше входили через Gmail, теперь входите по той же почте и паролю, без кнопки Gmail.

  • Informatics: запасной вход

    Используйте, если основной адрес не открывается. Вход тот же: по почте и паролю, без кнопки Gmail.
