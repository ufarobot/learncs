Internal system

Дизайн-система лендингов learncs.ru

Эта страница фиксирует токены, блоки и рецепты, чтобы новые страницы собирались из существующих паттернов, а не из разовых решений.

Главная Рецепты

Tokens

Базовые решения

Цвет, радиус, фон и CTA должны приходить из токенов. Новый hard-coded цвет считается исключением и должен попасть в гайд.

White background

--bg для основной страницы.

Soft card surface

--card-bg для карточек и summary-блоков.

learncs blue

--blue только для главного действия.

Ink text

--ink для основного текста и заголовков.

Components

Разрешенные блоки

Блоки ниже являются словарем для новых лендингов. Если нужен новый визуальный паттерн, сначала добавляем его сюда.

Buttons

CTA и вторичные действия

Главная кнопка всегда синяя и прямоугольная с мягким радиусом. Вторичные действия нейтральные.

Оставить заявку Программа
Feature grid

Карточки преимуществ

Нейтральная поверхность, компактный заголовок, короткий текст. Без теней, без декоративных градиентов.

  • одно сильное сообщение на карточку;
  • 2-4 карточки в сетке;
  • изображение сверху, если оно есть.
Media card

Image-led карточка

Фотография или скриншот несет визуальный вес. По умолчанию не кропаем изображение.

Короткие факты

Recipes

Рецепты страниц

Новая страница начинается с рецепта. Меняем содержание и порядок только при понятной продуктовой причине.
