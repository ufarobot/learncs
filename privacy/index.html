Политика обработки персональных данных

Последнее обновление: 2 июня 2026 года.

1. Оператор

Оператор персональных данных: Ишимбаев Айрат Наилевич, автор и преподаватель курса learncs.ru. Для обращений по вопросам обработки персональных данных можно написать в Telegram: @aishimbaev.

2. Какие данные обрабатываются

Через форму заявки сайт может получать имя, контакт для связи и адрес страницы, с которой отправлена заявка. При использовании Яндекс Метрики также могут обрабатываться технические данные о посещении сайта: источник перехода, параметры URL, действия на странице, сведения о браузере и устройстве.

3. Цели обработки

  • связаться с пользователем по заявке на занятия;
  • ответить на вопросы о курсе, расписании и формате обучения;
  • анализировать работу сайта и эффективность рекламных кампаний;
  • улучшать содержание сайта и форму заявки.

4. Правовое основание

Обработка выполняется на основании согласия пользователя, выраженного при отправке формы заявки, а также в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

5. Как обрабатываются данные

Оператор может выполнять сбор, запись, хранение, уточнение, использование и удаление персональных данных. Данные используются только для связи по заявке, организации обучения и анализа работы сайта.

6. Сервисы и передача данных

Для приема заявок сайт использует внешний технический endpoint Google Apps Script. Для веб-аналитики используется Яндекс Метрика. Если пользователь пишет напрямую в Telegram или MAX, дальнейшая переписка ведется в выбранном пользователем мессенджере по правилам соответствующего сервиса.

7. Срок хранения

Данные заявки хранятся до достижения цели обращения или до отзыва согласия пользователем, если более длительный срок не требуется по закону или для защиты прав оператора.

8. Права пользователя

Пользователь может запросить информацию об обработке своих персональных данных, попросить уточнить, удалить или заблокировать данные, а также отозвать согласие на обработку. Для этого нужно обратиться к оператору по контакту, указанному в разделе 1.
