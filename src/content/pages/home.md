# Редакторский просмотр главной

Этот файл хранит контент главной страницы. Основной источник данных находится в YAML-блоке выше: `blocks[]` собирает страницу из готовых layout-шаблонов. Визуальную композицию задают Astro-компоненты и общий CSS.

В WebStorm удобно редактировать этот файл рядом с локальным preview `http://127.0.0.1:8123/`. Markdown-preview полезен для чтения заметок и длинной редакторской версии, но финальный внешний вид проверяется именно через Astro preview.
